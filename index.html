Secure route origination

November 1, 2012

If you would like to learn more about what you can do to help secure BGP routing on the Internet, check out our RPKI documentation. We also offer a free RPKI browser that allows you to view the essential information inside the X509 certificates, CRLs, manifests and ROAs that can be found in an RPKI repository. It can be found at rpki.cobenian.com. This is a bare bones application that was developed in under 24 hours. More features will be added over time.
